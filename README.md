# Soccer



### サイトテーマ
サッカーをするメンバーを集めるサイト
・サッカーをしたいがメンバーがなかなか集まらない
・引越しや環境の変化で友達ができない
・大会運営者の参加チームが集まらない
などを解決できる人集めサイト

：サッカーをやりたい方のコミュニティ、サッカーをするための人数集め、場所確保するコミュニティサイトです。


### テーマを選んだ理由
地元にいた時は、友達とよく集まりサッカーをしていたが引っ越してからなかなか友達ができずサッカーをできずにいたので、人が集まり試合ができるサイトがあればと思い選びました。
開催側も参加チームの確保に困っているとの声を聞いた為このテーマにしました。


### ターゲットユーザ
- フットサルやサッカー場の管理者
- サッカースクールの人
- 決まった日にちしか休みのない社会人
- とにかく体を動かしたい人
- サッカーしたい大学生
- イベント開催者
１０代から４０代


### 主な利用シーン
- サッカーをする際にメンバーを集める時
- 参加チーム数を増やし大会を盛り上げたい時
- サッカースクールの入会者集め
- 友達を作りたい
- 体を動かしたい
